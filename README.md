# iOS-Budget-Tracker
A native iOS application built with SwiftUI that tracks spending, automates KPIs, and visualizes data using Swift Charts.

Vibe Expense Tracker
A native iOS expense tracking application built with SwiftUI.  
Key Features
Personalized Budgeting: Onboarding flow to set custom monthly spending limits.  
Transaction Management: Add, track, and categorize daily expenses in real-time.  
Data Visualization: Uses Swift Charts to provide intuitive "Vibe Recap" analytics, including category breakdowns and spending flow charts.  
How to Use
This app is designed to be paired with an iOS Shortcut to be fully functional. To utilize the automation feature:
Open the iOS Shortcuts app.  
Create a new shortcut that captures your expense data (Merchant, Amount, Category).  
Technical Stack
Frameworks: SwiftUI, Combine, Charts.  
Data Persistence: UserDefaults for local storage.  
Architecture: MVVM pattern with ObservableObject for state management.
